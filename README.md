# 📊 Loan Risk Analytics Dashboard

## 🔍 Project Overview
This project presents a multi-page interactive Power BI dashboard designed to analyze loan performance, borrower demographics, and financial risk indicators.

The dashboard simulates a banking risk analytics environment to support data-driven decision making.

---

## 🛠 Tools & Technologies
- Power BI
- DAX
- SQL Server
- Power Query
- Excel

---
