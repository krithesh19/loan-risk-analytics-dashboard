# 📊 Loan Risk Analytics Dashboard – Banking Risk Intelligence (Power BI)

## 📌 Project Overview

Designed and developed a multi-page interactive Power BI dashboard simulating a banking loan portfolio risk analytics system.

The solution analyzes loan performance, borrower demographics, credit risk indicators, and time-based portfolio trends to support data-driven lending decisions and proactive risk monitoring.

This project demonstrates structured BI development, financial analytics, and decision-support dashboard design in a simulated banking environment.

---

## 🛠 Tools & Technologies

- Power BI  
- DAX (Time Intelligence & Risk Metrics)  
- SQL Server  
- Power Query (ETL)  
- Microsoft Excel

# 🎯 Business Objectives

- Monitor overall loan portfolio performance  
- Identify high-risk borrower segments  
- Track default rate trends over time  
- Evaluate credit exposure across demographics  
- Support strategic lending and credit policy decisions  

---

# 🏗 Data Modeling & Architecture

- Integrated loan data using SQL Server  
- Performed ETL transformations using Power Query  
- Built structured data model optimized for analytical reporting  
- Created reusable DAX measures for KPI standardization  
- Applied time-intelligence functions for trend analysis  

The model enables scalable risk monitoring and portfolio performance tracking.
